##### &nbsp;                                   **VPN Setup and Privacy Analysis Report**







###### **Tool Used**



* ProtonVPN (Free Version)
* Operating System: Windows
* IP Verification Website: https://whatismyipaddress.com





###### **Step-by-Step VPN Setup Process**



###### **1. Choose a VPN Service**



I selected ProtonVPN Free Tier because it is reputable and offers secure encryption without requiring payment.



###### **2. Download and Install**



* Visited the official ProtonVPN website.
* Downloaded the Windows client.
* Installed the application following setup instructions.



###### **3. Create an Account**



* Signed up with email and password.
* Verified email address.



###### **4. Connect to a VPN Server**



* Opened the ProtonVPN app.
* Logged in.
* Selected the closest available free server location.
* Clicked Connect.
* Status changed to Connected.



###### **5. Verify IP Address Change**



* Before VPN: Checked IP using whatismyipaddress.com.
* After VPN connection: Refreshed the page.
* The IP address and location changed, confirming VPN was working.



###### **6. Test Secure Browsing**



* Opened HTTPS websites.
* Verified secure connection (lock symbol in browser).
* Browsing traffic was encrypted through VPN tunnel.



###### **7. Disconnect VPN**



* Clicked Disconnect.
* Checked IP again.
* IP returned to original address.
* Compared browsing speed (VPN was slightly slower than normal connection).







###### **VPN Encryption \& Privacy Features**



* AES-256 Encryption – Protects data from interception.
* Secure VPN Protocols (OpenVPN / WireGuard).
* IP Masking – Hides real IP address.
* Kill Switch (if available) – Blocks traffic if VPN disconnects.
* No-logs Policy (depending on provider)







###### **Benefits of Using a VPN**



* Protects data on public Wi-Fi
* Hides real IP address
* Encrypts internet traffic
* Helps bypass geo-restrictions
* Improves online privacy







###### **Limitations of VPN**



* May reduce internet speed
* Free versions have limited servers
* Does not provide complete anonymity
* Some websites block VPN traffic







###### **Conclusion**



This task provided hands-on experience in setting up and using a VPN.

The experiment demonstrated how VPNs encrypt traffic, mask IP addresses, and enhance online privacy.

However, performance impact and feature limitations should be considered when using free VPN services.
